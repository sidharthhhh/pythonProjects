# AI Image Authenticity & Forensics Platform

A production-grade platform for detecting AI-generated images through a multi-layer pipeline:
- Cryptographic provenance (C2PA)
- Invisible watermark detection
- Deep-learning classification
- Metadata analysis
- Advanced signal forensics

## Getting Started

This directory will contain the source code for the platform, which exposes a FastAPI backend for image analysis.

Please refer to the architecture plan for more details.
